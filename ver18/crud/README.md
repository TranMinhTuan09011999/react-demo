1.Setup reactjs
### `npx create-react-app reactdemo`
### `cd reactdemo`
### `npm i react-router-dom`
### `npm i react-axios`
### `npm i @material-ui/core`
### `npm i json-server`
### `npm start` // run the project
**Note: Should have latest node version installed on our system**

2.Run json server inside your terminal
### `yarn json-server`
